Build Local AI Assistant with PHP-8 & Ollama
--------------------------------------------
Stacks:
- PHP-8.x & SQLite
- Ollama
- AI Models (gemma3:1b, qwen2.5:0.5b, embeddinggemma);
- HTMX, AlpineJS, Tailwind-CSS
- PWA Installer
Dev System: 
- OS: Mac / Linux
- Browser : Google Chrome


##############################################################################
Step 1 (DONE)
====================
Basic :
    - Build custom Model
====================

    Install Ollama :
    ----------------
    https://ollama.com/


    Add Main Model:
    --------------
    ollama pull gemma3:1b


    Build Directory Structure:
    --------------------------
    mkdir asisten-chat src


    Basic Model Files:
    ------------------
    touch asisten-chat/Modelfile-Default


    Create Custom Model:
    ---------------------
    ollama create default-chat -f ./asisten-chat/Modelfile-Default

    Basic Testing :
    ---------------
    touch src/OllamaExec.php test.php

    Run Test:
    ---------
    php test.php


##############################################################################
Step 2 (DONE)
Basic view with HTMX
====================
Version 1 :
    - Build Basic Frontend HTMX Index V1
    - Build backend to handle AJAX
====================

    New Variation Model File:
    -------------------------
    mkdir agen-koding && 
    touch asisten-chat/Modelfile-Riset && 
    touch agen-koding/Modelfile-Coder


    Create Another Custom Models:
    https://docs.ollama.com/modelfile
    -----------------------------
    ollama create asisten-riset -f ./asisten-chat/Modelfile-Riset
    ollama create agen-koding -f ./agen-koding/Modelfile-Coder


    Build Assets Directory Structure:
    ---------------------------------
    mkdir -p public/assets/{js,css}


    Index & assets files:
    ---------------------
    touch public/index.php public/api.php src/helpers.php public/bootstrap.php
    download assets from CDN

    Run PHP Built-in web server
    ---------------------------
    
    php -S localhost:8000 -t public
    or (Publish to Local Network)
    php -S 0.0.0.0:8000 -t public

    Prompt :
    --------
    Chat :
    tulis Halo
    Kode:
    tulis Halo dengan program PHP
