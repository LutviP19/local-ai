Build Local AI Assistant with PHP-8 & Ollama
--------------------------------------------
Stacks:
- PHP-8.x & SQLite
- Ollama
- AI Models (gemma3:1b, qwen2.5:0.5b, embeddinggemma);
- HTMX, AlpineJS, Tailwind-CSS
- PWA Installer
Dev System: 
- OS: Mac / Linux
- Browser : Google Chrome


##############################################################################
Step 1 (DONE)
====================
Basic :
    - Build custom Model
====================

    Install Ollama :
    ----------------
    https://ollama.com/


    Add Main Model:
    --------------
    ollama pull gemma3:1b


    Build Directory Structure:
    --------------------------
    mkdir asisten-chat src


    Basic Model Files:
    ------------------
    touch asisten-chat/Modelfile-Default


    Create Custom Model:
    ---------------------
    ollama create default-chat -f ./asisten-chat/Modelfile-Default

    Basic Testing :
    ---------------
    touch src/OllamaExec.php test.php

    Run Test:
    ---------
    php test.php

Integating with local system and network
====================
Extra 
Install PWA to StartUp Application :
    - Enhance Index more clean, stable and interactive : Index V3 (Final)
    - Setting up local network to allow access LocalAI
    - Install Application to system / mobile phone
    - Add command and installed PWA at StartUp
====================

    Command to Find bin path of PHP (Mac / Linux):
    -: which php

    1. Add run custom command at StartUp:
        <php bin path> -S localhost:8000 <full_path_to_project>
        eg:
        /usr/bin/php -S 0.0.0.0:8000 /home/my/project_path/public

    2. Add Installed PWA at StartUp Application.

    3. Restart OS.
